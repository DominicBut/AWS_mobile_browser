CUSTOMIZATION
==============

To change the default AWS portal URL:

1. Open index.html in a text editor
2. Find this line:
   const AWS_PORTAL = 'https://internal.awsapps.com/start';
3. Replace with your domain
4. Save the file

To change colors:

1. Open index.html in a text editor
2. Find the CSS section (inside <style> tags)
3. Change these values:
   ‚Ä¢ #FF9900 = AWS Orange (main color)
   ‚Ä¢ #232f3e = AWS Dark (secondary color)
4. Save the file
